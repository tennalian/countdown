Countdown.js
=======================

Initialization
------------

```code
    <script src="js/countdown.js"></script>

    <script>
        countdown('.countdown');
    </script>
```


Options
------------

Light template

```code
	<div class="countdown" data-style='light'></div>
```

Set time

```code
	<div class="countdown" data-time="Jan,30,2016 00:00:00"></div>
```
